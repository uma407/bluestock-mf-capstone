# Bluestock Mutual Fund Analytics Platform

## Project Overview

This project is an end-to-end Mutual Fund Analytics Platform developed as part of the Bluestock Data Analyst Internship Capstone Project. The platform performs data ingestion, ETL processing, database management, exploratory data analysis, performance analytics, advanced risk analytics, and interactive dashboard reporting using real mutual fund datasets.
